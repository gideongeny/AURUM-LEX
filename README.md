## üöÄ Deployment

This site is static and ready for immediate deployment on any modern host.

### Netlify (Recommended)
1.  Drag and drop this folder into [Netlify Drop](https://app.netlify.com/drop).
2.  Your site will be live in seconds.

### Vercel
1.  Install Vercel CLI: `npm i -g vercel`
2.  Run `vercel` in this directory.

## ‚öñÔ∏è Customization

To personalize for your firm:
1.  **Contact Form**: Update the `action` attribute in the `<form>` tag in `index.html` with your own [Formspree](https://formspree.io) endpoint.
2.  **Schema**: Edit the `<script type="application/ld+json">` block in the `<head>` of `index.html` with your specific address and phone number.
